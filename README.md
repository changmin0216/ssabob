# 냠냠코치 (YumYumCoach)

## 📌 프로젝트 개요

**냠냠코치(YumYumCoach)** 는 식단 기록 및 영양 분석을 기반으로 한 개인 맞춤형 건강 코칭 서비스입니다.  
사용자가 직접 식단을 기록하고, 영양 성분을 분석하여 건강 목표(다이어트, 운동, 질병 관리 등)에 맞는 식단 관리와 챌린지를 지원합니다.  
또한 지도 기반 기능을 활용하여 내 주변의 샐러드 가게를 쉽게 찾을 수 있도록 제공합니다.

| ------ | ----------------------------- | ----- | -------------------------------------------- | ---- |
| 기본   | 식단기록 관리 기능            | 30    | 화면, 등록, 조회, 수정, 삭제 기능            | ✅   |
| 기본   | 회원정보 관리 기능            | 30    | 화면, 등록, 조회, 수정, 삭제 기능            | ✅   |

| 심화   | 팀별 아이디어                 | 10    | 타 사이트 벤치마킹 또는 새로운 아이디어 구현 |      |
| 총점   | ----------------------------- | ----- | ---------------------------------------      | 60   |

### 1. 식단 관리 기능

- 식단 기록 작성, 조회, 수정, 삭제 (CRUD)
- 섭취한 음식별 영양 성분

### 2. 회원가입 및 로그인 기능

- 회원가입을 통해 사용자 정보(아이디, 비밀번호, 이름 등) 등록
- 로그인 / 로그아웃 기능 구현
- 회원 수정, 삭제 기능 구현
